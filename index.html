MAPA WEB INTERACTIVO USANDO GEOTECNOLOGÍAS LIBRES

Para disponer de un mapa web no es necesario comprar una licencia de ArcGIS Online, solo son necesarias las interacciones de conocimientos alternativos: "uso de datos en GeoPackage, uso del software QGIS, y usos básicos de HTML5 y CSS3", tales interacciones resultarán en un mapa web interactivo que permitirá a las jefaturas, entender, interpretar y decidir sobre los avances del mapeo de diferentes variables (capas) en beneficio de la entidad.
